# Security Policy

## Reporting a vulnerability
Please do not disclose potentially sensitive security findings in a public issue.

If you find a security concern:
- Contact the maintainer through this repository's GitHub security channel.
- Include:
  - a clear description of the issue
  - steps to reproduce
  - impact assessment
  - suggested mitigation, if known

## Scope
This is a non-commercial open-source project. Security reports are handled on a
best-effort basis by the maintainer.
