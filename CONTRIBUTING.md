# Contributing to video_editing_ai

Thanks for checking this out.

## Scope
This repository is a non-commercial open-source project for learning and experimentation
in AI-driven short-form video workflows.

## How to contribute
1. Open an issue describing your bug, idea, or enhancement.
2. Fork the repository.
3. Create a branch with a clear name.
4. Make a small, focused change.
5. Run available checks before opening a pull request.
6. Open a pull request with:
   - what changed
   - why it changed
   - how to validate it

## Development setup
Use the project-specific setup commands shown in the README and relevant module README
files. As a minimum baseline:

```bash
# clone
git clone https://github.com/prrcarvalho/video-editing-ai.git
cd video-editing-ai
```

## Pull request expectations
- Keep PRs focused and small.
- Do not commit secrets, `.env` files, generated videos, or model weight files.
- Include tests/checks notes for any behavior change.
- Add or update docs when behavior or run requirements change.
