# SFX Library

A set of short, lossless sound effects (WAV, PCM 16-bit, 44.1kHz, stereo) for use in video compositions.

This README is written for AI agents: each clip lists what it **sounds like** and **when to use it**, so you can pick the right effect from its description without listening. `manifest.json` holds the same data in machine-readable form.

## Clips

- **01_Gear.wav** — `0.44s`
  - Sound: a short mechanical gear turn / cog ratchet with a metallic, machine-like grind.
  - Use for: opening hook, machine-start cue, mechanical emphasis.

- **02_Metallic_Riser.wav** — `1.69s`
  - Sound: a rising metallic sweep that builds in pitch and intensity toward a peak.
  - Use for: tension build, reveal ramp, "wait for it" lead-in.

- **03_Click_Mouse.wav** — `0.12s`
  - Sound: a single crisp mouse-button click, dry and snappy.
  - Use for: UI tap, app demo, cursor click, cut-point accent.

- **04_Keyboard_Typing.wav** — `1.00s`
  - Sound: a quick burst of mechanical keyboard keystrokes.
  - Use for: building-in-public montage, coding shot, productivity vibe.

- **05_Wind_Woosh.wav** — `1.09s`
  - Sound: an airy wind whoosh that sweeps past, soft attack with a fast decay.
  - Use for: fast motion transition, swipe cut, scene change.

- **06_Pop.wav** — `0.08s`
  - Sound: a tiny, bright pop / bubble burst, very short.
  - Use for: micro punchline, sticker/emoji pop, tiny reveal.

- **07_Snap_Sound.wav** — `1.02s`
  - Sound: a sharp snap hit with a short resonant tail.
  - Use for: hard emphasis, reaction beat, clean transition hit.

- **08_Metallic_Woosh.wav** — `1.17s`
  - Sound: a slick metallic whoosh sweeping by, futuristic and polished.
  - Use for: premium product reveal, futuristic sweep, slick motion graphics.

- **09_Camera_Flash.wav** — `1.48s`
  - Sound: a camera-flash charge-and-pop with a bright shutter snap.
  - Use for: before/after reveal, thumbnail moment, freeze-frame ending.
