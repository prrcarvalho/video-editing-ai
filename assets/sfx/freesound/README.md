# Freesound Downloads

Original Freesound assets downloaded through `scripts/freesound_download.py` live here.

Agents should not place anonymous audio files in this directory. Every downloaded file must have a corresponding entry in `manifest.json` with:

- Freesound sound id
- source URL
- uploader username
- license
- attribution requirement
- intended usage
- selection note

Use `docs/mcp/freesound.md` for the full search/download workflow.
